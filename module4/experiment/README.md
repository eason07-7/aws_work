# module4

> generic 專題 — 使用 [workflow](https://github.com/) skill 起家。

## Quick Start

```bash
# Clone 回家後
cd module4

# 開 Claude Code，它會自動讀 CLAUDE.md + PROJECT_STATE.md
```

告訴 Claude 你的目標即可開工。

## 專題結構

- `CLAUDE.md` — 給 Claude 的規則（必讀）
- `PROJECT_STATE.md` — 一頁狀態板
- `JOURNAL.md` — 關鍵決策日誌（SSOT）
- `reference_code/` — 放老師/同事/網路的參考碼
- `_ppt_workspace/` — PPT 素材（邊做邊丟）

## 產出

完成後執行：

```bash
python D:\p\workflow\scripts\build_ppt.py .
```

產生 `_ppt_workspace/output.pptx`。
